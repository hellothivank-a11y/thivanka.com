# Oasis — Private Couple Space

Oasis is a premium, secure, and end-to-end encrypted web application designed specifically for couples to chat and share media in their own private bubble.

Built with **HTML5, CSS3, JavaScript (ES6), Supabase, CryptoJS, and PeerJS**.

## Features

*   **End-to-End Encryption (E2EE)**: All texts, images, and audio messages are encrypted locally using AES-256 on the client-side before leaving the browser. The database only stores ciphertext.
*   **Zero-Schema Media Sharing**: Custom compression and client-side encryption of image files and voice notes allows media sharing using standard database text columns. No complex storage buckets required.
*   **Emoji Safety Code**: SHA-256 fingerprint generated from your secret key maps to 5 emojis. Verify that both you and your partner see the exact same emojis to guarantee your keys match.
*   **Private Calling**: Secure peer-to-peer audio and video calling powered by PeerJS. Includes a ringing incoming call modal and in-call toggle switches for muting audio and video.
*   **Panic Button**: Pressing `Escape` instantly locks the application, wipes decrypted text from the viewport, and clears session credentials from memory.

## Database Setup

To run this application, you will need a Supabase database instance. In your Supabase dashboard, open the **SQL Editor** and run the following queries to create the necessary tables:

```sql
-- 1. Create the Chat Messages table
CREATE TABLE chat_messages (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    sender TEXT NOT NULL,
    encrypted_message TEXT NOT NULL,
    created_at TIMESTAMPTZ DEFAULT NOW() NOT NULL
);

-- Enable realtime changes for chat_messages table
ALTER PUBLICATION supabase_realtime ADD TABLE chat_messages;

-- 2. Create the Online Status table
CREATE TABLE online_status (
    username TEXT PRIMARY KEY,
    last_seen TIMESTAMPTZ DEFAULT NOW() NOT NULL
);
```

### Config Custom Keys (Optional)
If you want to use your own Supabase instance, update the constants at the top of [app.js](app.js):
```javascript
const SUPABASE_URL = 'YOUR_SUPABASE_URL';
const SUPABASE_ANON_KEY = 'YOUR_SUPABASE_ANON_KEY';
```

## Deploying to `thivanka.com/chat` using GitHub Pages

Because Oasis is a client-side serverless application, you can host it for free on **GitHub Pages**!

### Step 1: Create a GitHub Repository
1.  Log in to your GitHub account.
2.  Create a new repository named `chat`.
3.  Make it **Public** (required for free GitHub Pages).

### Step 2: Upload Files to GitHub
Since `git` commands might require developer tools on your Mac, the easiest way to upload is through the GitHub web interface:
1.  In your new repository on GitHub, click **"uploading an existing file"** in the setup box.
2.  Drag and drop the following files into the browser:
    *   `index.html`
    *   `style.css`
    *   `app.js`
    *   `README.md`
    *   `.gitignore`
3.  Click **Commit changes**.

*(Alternatively, if you have Git installed, run:)*
```bash
git init
git add .
git commit -m "Initial commit of Oasis Chat Space"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/chat.git
git push -u origin main
```

### Step 3: Configure Custom Domain on GitHub Pages
To serve this repository at your custom domain `thivanka.com/chat`:
1.  Go to your repository **Settings** tab on GitHub.
2.  Scroll down to the **Pages** section in the left sidebar.
3.  Under **Build and deployment**, set the source to **Deploy from a branch** and select the branch `main` (folder `/root`), then click **Save**.
4.  To host under your custom domain `thivanka.com`:
    *   Under **Custom domain**, enter `thivanka.com`.
    *   Click **Save**.
    *   *(Note: Ensure your DNS settings for `thivanka.com` are pointing to GitHub Pages servers as described in [GitHub custom domains documentation](https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site).)*

Once configured, GitHub will deploy your site. Since the repository is named `chat`, it will automatically be available at `https://thivanka.com/chat/`.
